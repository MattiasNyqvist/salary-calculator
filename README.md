# Salary Analyzer

En enkel löneanalys-app byggd med Streamlit och Pandas.

## Funktioner

- Översikt av lönestatistik
- Analys per avdelning
- Outlier detection
- Sökfunktion

## Kör appen
```bash
streamlit run app.py
```

## Lärdomar från detta projekt

- Pandas för dataanalys
- Streamlit för UI
- Plotly för visualiseringar
- Basic statistik (mean, median, std)
